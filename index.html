Things you need to know




#1 node is single threaded

!

...it's better than it sounds

Your code runs in a single thread

All IO operations happen asynchronously

When you call an IO operation, you're adding an event to an event queue
Node runs an 'event loop'
  1. that picks up an event
  2. processes it
  3. when it's done, calls your callback

...and you can create 'clusters' of nodes


    var cluster = require('cluster');

    if (cluster.isMaster) {
        //start up workers for each cpu
        require('os').cpus().forEach(function() {
            cluster.fork();
        });

    } else {
        //load up your application as a worker
        require('./app.js');
    }
